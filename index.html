How to Play

Each guess reveals how close your stratagem is to today's secret one.

Green = Exact match
Red = Incorrect category
⬆️ / ⬇️ = Higher or lower (for numbers like Cost or Cooldown)

Categories

  • Department: Where the stratagem comes from (e.g. Hangar, Patriotic Administration Center, Warbond).
  • Type: Support, Offensive, Defensive, Common or Objectives
  • Arrows: Number of arrow inputs in its code.
  • Level: Unlock level (0 if always available).
  • Cooldown: Time before it can be used again. For Eagle Strikes, this refers to the re-arm time.
  • Call-in Time: Time it takes to land after calling. Measured pre-upgrade.
  • Limited Uses: If the stratagem works on a charges system like eagle strikes or reinforcements. True or False
    • This includes Eagle Strikes and similar expendable abilities.
